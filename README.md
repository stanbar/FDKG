[Presentation of the protocol](https://stan.bar/slides/2023-10-22-A-Voter-to-Voter-voting-protocol.pdf)

1. Initialise submodules with:
`git submodule update --init --recursive`
2. Generate circuits
Go to the circuits folder with:
`cd circuits`
